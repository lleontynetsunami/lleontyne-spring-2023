<html> Construction

Structure

We will be using the structure we went over in the last lecture.

Please note that the majority of the tags we will cover in this lecture need a closing tag </>

<header> tag

The <header> element represents a container for introductory content or a set of navigational links. You can have several <header> elements in one document. A <header> tag cannot be placed within a <footer>, <address> or another <header> element.

Don't confuse this with the <header> tag, the <head> tag lives outside of the body tag.

<nav> tag

The <nav> tag defines a set of navigation links. Notice that NOT all links of a document should be inside a <nav> element. The <nav> element is intended only for major block of navigation links.

Browsers, such as screen readers for disabled users, can use this element to determine whether to omit the initial rendering of this content.

<section> tag

The <section> tag defines sections in a document, such as chapters, headers, footers, or any other sections of the document.

The <section> tag defines sections in a document. You can have multiple <section> tags with in a document. You could even nest <section> tags.

<article> tag

The <article> tag specifies independent, self-contained content. An article should make sense on its own and it should be possible to distribute it independently from the rest of the site

You will see the <article> tag and <section> tag substituted for each other.

<aside> tag

The <aside> tag defines some content aside from the content it is placed in. The aside content should be related to the surrounding content.

The aside content is the same content that you would use for an aside in your publication class.

<div> tag

The <div> tag defines a division or a section in an HTML document.The <div> tag is used to group block-elements to format them with CSS. We will ues the <div> tag in connection with CSS Grid.

The <div> tag is a tag that has been abused over the years. With the introduction of new tags in HTML the div tag now has a very specific purpose.

<footer> tag

The <footer> tag defines a footer for a document or section. A <footer> element should contain information about its containing element.

You can have several <footer> tags in one document.
