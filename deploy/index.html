<html> Introductions

Basic HTML page structure

Document Type

The declaration must be the very first thing in your HTML document, before the <html> tag. Notice how much cleaner the html 5 doctype is vs. the html 4.

HTML 5 DOCTYPE

Please note we will always use HTML 5 docs for this class and all future web classes.

<!DOCTYPE html>

HTML 4 DOCTYPE

<!DOCTYPE HTML PUBLIC-//W3C//DTD HTML 4.01 Transitional//EN" "http://www.w3.org/TR/html4/loose.dtd>

<html> tag

Defines the root of an HTML document.

The <html> tag tells the browser that this is an HTML document.

The <html> tag is the container for all other HTML elements (except for the <!DOCTYPE> tag).

<head> tag

The <head> tag in HTML is used to define the head portion of the document which contains information related to the document.

Do not confuse this for the <header> tag

<meta> tag

The <meta> tag provides metadata about the HTML document. Metadata will not be displayed on the page, but will be machine parsable.

Meta elements are typically used to specify page description, keywords, author of the document, last modified, and other metadata.

Does not have a closing tag.

<title> tag

Provides a title for the page when it is added to favorites. Displays a title for the page in search-engine results

<body> tag

The <body> tag defines the document's body.

The element contains all the contents of an HTML document, such as text, hyperlinks, images, tables, lists, etc.
